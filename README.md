# Firebird Computing
### Sovereign AI Infrastructure — Built in Africa

This repository contains the codebase for the Firebird Computing platform.

Project Structure
```/frontend     → Next.js web application```
```/backend      → API and server components```

Getting Started
Frontend
```
    cd frontend
    npm install
    npm run dev
```

Backend
```
    cd backend
    npm install
    npm start
```
